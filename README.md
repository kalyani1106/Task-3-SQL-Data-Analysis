# Task 3 - SQL for Data Analysis

## Objective

To use SQL queries to extract, manipulate, and analyze data from a structured database.

## Tool Used

* DB Browser for SQLite
* SQL

## Dataset

Sample Superstore Dataset

## Database

The dataset was imported into SQLite and stored as a table for analysis.

## SQL Concepts Covered

* SELECT
* WHERE
* ORDER BY
* GROUP BY
* HAVING
* Aggregate Functions (SUM, AVG)
* Subqueries
* Views
* Indexes

## Queries Performed

### 1. SELECT Statement

Retrieved records from the dataset.

### 2. WHERE Clause

Filtered records based on sales conditions.

### 3. ORDER BY

Sorted data in descending order of sales.

### 4. GROUP BY

Analyzed total sales by category.

### 5. HAVING Clause

Filtered grouped results based on total sales.

### 6. Aggregate Functions

Calculated total sales and average sales values.

### 7. Subquery

Retrieved records with sales greater than the average sales value.

### 8. View Creation

Created a reusable view for category-wise sales analysis.

### 9. Index Creation

Created an index on the Region column to improve query performance.

## Key Insights

* Technology category generated the highest sales.
* Sales distribution varied significantly across categories.
* Several orders had sales values higher than the overall average.
* Category-wise aggregation provided useful business insights.
